    Takeaways - Errors

    Cause of user failure

    Rarely due to lack of documentation. More often:

    • Not sure how to fix an error
    • Not sure how to solve their own problem
    • Couldn't find the right docs
    • Model of the product does not match your model of the product

    Making it better

    • Figure out the common problems users have
    • Make better error messages, if you can
    • Or solve the problem for the user
